# Tic-Tac-Toe Game (C# Windows Forms)

An interactive, desktop-based **Tic-Tac-Toe** game built using **C#** and **Windows Forms**. This project demonstrates core programming concepts, including Object-Oriented Programming (OOP), event-driven architecture, and dynamic UI updates.

---

## 🚀 Features

* **Two-Player Mode:** Play locally alternating between 'X' and 'O'.
* **Win/Draw Detection:** Instant automated validation after every click to check for rows, columns, or diagonals.
* **Custom Assets:** Interactive UI utilizes graphical image resources (`X.png`, `O.png`) for a polished user experience.
* **Score & Reset Tracker:** Easy reset mechanism to restart the match immediately without restarting the application.

---

## 🛠️ Technologies Used

* **Language:** C#
* **Framework:** .NET Framework (Windows Forms)
* **IDE:** Visual Studio

---

## 📂 Project Structure

A brief look at the core repository files:
* `Form1.cs` - Contains the core game logic, turn management, and board validation algorithms.
* `Program.cs` - The main entry point for the application.
* `Resources/` - Holds the graphical assets used for the game grid and players.
